## Self-hosting

### From Repo
1. **Clone the repo**

    Clone it on your local machine using:
    ```bash
    git clone https://github.com/boon4681/munbun.git
    ```

2. **Install Dependencies**
   
    ```bash
    yarn
    # or
    npm install
    ```


3. **Copy the env.example file into .env**
   
    ```bash
    cp .env.example .env
    ```

4. **Run it**

    ```bash
    yarn dev --host
    ```
